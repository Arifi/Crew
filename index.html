Crew by Team Fusion

Code review tool for Git projects.

Crew is a code review tool for git projects.
It works with a simple API.
So, you can easily integrate it in your workflow.

Crew is a standalone LAMP project.
It is based on web common technologies : symfony 1.x, jQuery and LESS.
So, you can easily install it on your own server.

Crew is an open-source project.
It is published under MIT license.
So, you can use it Like A Boss !
